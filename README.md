# Retail-Sales-Dashboard
This project is an interactive Power BI dashboard designed to analyze retail sales performance across different regions, product categories, and customer types.

# Objectives
1. Analyze total revenue, orders, and quantity sold
2. Identify top-performing regions and product categories
3. Understand customer purchasing behavior
4. Track sales trends over time

## Tools & Technologies

1. Power BI Desktop
2. Microsoft Excel
3. DAX (Data Analysis Expressions)

# Key Insights

1. Gauteng generates the highest revenue among all regions
2. Electronics is the top-performing product category
3. Returning customers contribute slightly more revenue than new customers
4. Sales show an upward trend into February despite fluctuations
